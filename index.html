Need a simple weather forcast to display?

PyWeather provides a simple interface for generating a simple, graphical weather forecast for display in HTML.

import forecastdata
import svgmanip

forecast = forecastdata.Forecast('CA/SanFrancisco')
svgmanip.write_forecast(forecast)
